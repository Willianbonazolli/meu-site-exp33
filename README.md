 Expedição 33 - Site Interativo
<br><br>
## "For those who come after"
Visão Geral <br><br>
O Site foi desenvolvido como um projeto experimental com tema inspirado no Jogo Expedition 33, apresentando uma experiência narrativa imersiva sobre o jobo. A ideia era trabalhar as habilidades de construção de site, semântica, indentação e Design. Fiz 3 páginas diferentes que seguem esse estilo: background estilizado com imagens, efeitos de loading e sobreposição, efeito canvas para geração de pétalas. Tive bastante trabalho em fazer o formato de pétalas por estar utilizando um método que envolve diversos cálculos.

<br>

Conceito <br><br>
O projeto explora um tipo storytelling digital através de:
Narrativa visual com elementos de mistério e aventura,
design e tipografia em uma jornada. Tentei aproximar um pouco do estilo do jogo também.
