 Expedição 33 - Site Interativo
<br><br>

Visão Geral <br><br>
Site desenvolvido como projeto experimental com tema inspirado no Jogo Expedition 33, apresentando uma experiência narrativa imersiva sobre uma expedição misteriosa.
<br><br>

Conceito <br><br>
O projeto explora um tipo storytelling digital através de:
Narrativa visual com elementos de mistério e aventura,
design e tipografia em uma jornada. Tentei aproximar um pouco do estilo do jogo também.
<br><br>

Tecnologias Utilizadas<br><br>
HTML5<br>
CSS3<br>
JavaScript <br>
